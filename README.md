## 프로젝트 개요

**Chronicles of Eternity**는 HTML5 Canvas 위에 순수 Vanilla JavaScript로 구현된 탑다운 뷰 던전 액션 RPG입니다.
빌드 도구·외부 프레임워크 없이 브라우저만으로 즉시 실행되며, 19종의 플레이어 캐릭터·8종의 몬스터·보스전·레벨업 시스템을 포함합니다.

### 핵심 특징

| 항목 | 내용 |
|------|------|
| 플레이어 캐릭터 | 19종 (남 10 / 여 9), 클래스별 고유 스킬 4개 |
| 몬스터 | 8종 + 보스 1종, 개별 AI 패턴 |
| 전투 | 근접/원거리/마법 공격, 크리티컬, 상태이상, 넉백 |
| 스테이지 | 절차적 생성 던전, 스테이지 클리어 시 다음 스테이지 진행 |
| 렌더링 | Canvas 2D, 60fps rAF 게임 루프, 카메라 추적, 미니맵 |
| 그래픽 | 이모지 기반 스프라이트 + 커스텀 벡터 캐릭터 렌더링 |

---

## 플레이 방법

1. 타이틀 화면에서 **새 게임** 클릭
2. 원하는 캐릭터를 선택하고 **시작** 클릭
3. 던전에서 몬스터를 처치해 보스 소환 조건 달성
4. 보스를 처치하면 스테이지 클리어 → 다음 스테이지로 진행

| 마법 | 마법 투사체 | 3방향 투사체 | 연쇄 마법 | 8방향 대형 마법탄 |

### 데미지 계산

```
피해 = max(1, 공격력 × 배율 - 방어력 × 0.3)
크리티컬 = 피해 × 2.3  (크리확률로 판정)
```

## 게임 시스템

### 레벨 & 경험치

- 기본 경험치 요구량: **120** (레벨업마다 1.4배 증가)
- 레벨업 스탯 증가: HP +10%, MP +8%, ATK/MATK +8%, DEF +6%

### 스테이지 구조

```
[스테이지 시작]
  ↓ 절차적 맵 생성 (38×30 타일, 벽 밀도 7%)
  ↓ 초기 몬스터 스폰: 14 + (스테이지 × 2)마리
  ↓ 처치 수 누적
  ↓ bossSpawnKills 도달: 14 + (스테이지 × 4)
  ↓ 보스 소환
  ↓ 보스 처치 → 스테이지 클리어
[다음 스테이지 or 타이틀]
```

### MP 재생

- 1.5초마다 최대 MP의 **1.5%** 자동 재생

### 상태이상

| 상태 | 원인 | 효과 |
|------|------|------|
| 독 (Poison) | 좀비 근접 공격 | 1초마다 독 피해 (서서히 감소) |
| 공격력 버프 | 음유시인 스킬 등 | ATK 배율 증가 (타이머 소진 시 해제) |
| 무적 (Invincible) | 피격 후 0.55초, 스킬 발동 시 | 모든 피해 무시 |

## 기술적 특징

### 렌더링 파이프라인

```
renderFrame()
  ├── renderMap()              — 타일 맵 (벽/바닥/데칼)
  ├── renderEffectsBehind()    — 바닥 레이어 이펙트
  ├── renderParticles()        — 물리 파티클
  ├── renderMonsters()         — 몬스터 이모지 + HP바
  ├── renderProjectiles()      — 투사체 + 궤적 트레일
  ├── renderPlayerV2()         — 벡터 캐릭터 + 무기 렌더링
  ├── renderEffectsFront()     — 전면 레이어 이펙트
  └── drawMinimap()            — 미니맵 Canvas
```

### 충돌 감지

- **타일 기반:** 4방향 코너 체크 (`wouldCollide`)
- **원 기반:** 플레이어 ↔ 몬스터, 투사체 ↔ 대상 (`dist < r1 + r2`)

### 카메라

```javascript
// 매 프레임 Lerp (13% 보간)
camera.x += (player.x - W/2 - camera.x) * 0.13;
camera.y += (player.y - H/2 - camera.y) * 0.13;
```

### 이모지 렌더링

- `bodyEmoji` + `weaponEmoji` 분리 렌더링
- 공격 시 무기 스윙 애니메이션 (각도 회전)
- 달리기 바운스, 무적 점멸 효과
